# CS 10-414: Deep Learning Systems

Personal study repository for **CMU 10-714/10-614 Deep Learning Systems** course.

## Course Information

This repository contains my coursework and learning materials for the Deep Learning Systems course (CS 10-714/10-614) at Carnegie Mellon University. The course focuses on the systems aspects of deep learning, covering:

- Automatic differentiation and computational graphs
- GPU programming and parallel computing
- Deep learning framework design and implementation
- Optimization algorithms and training techniques
- Convolutional and recurrent neural networks
- Model deployment and performance optimization


## Structure
